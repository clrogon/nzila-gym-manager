## Staff Roles | Funções de Staff

### EN

Each role has specific permissions and responsibilities:

| Role | Permissions | Key Responsibilities |
|------|-------------|---------------------|
| **Coach** | Create classes, manage workouts, track member progress | Design training programs, lead classes, assess skills |
| **Trainer** | Assign workouts, track member progress | Personal training sessions, one-on-one coaching |
| **Instructor** | Lead assigned classes | Teach group classes, record attendance |
| **Physiotherapist** | View health data, member recovery | Injury assessment, rehabilitation tracking |
| **Nutritionist** | Diet plans, member consultations | Nutrition guidance, meal planning |
| **Receptionist** | Check-in/out, payments, registration | Front desk operations, member assistance |
| **Staff** | Limited access, general operations | Supporting roles, facility maintenance |

### PT

Cada função tem permissões e responsabilidades específicas:

| Função | Permissões | Responsabilidades Chave |
|---------|------------|----------------------|
| **Treinador** | Criar aulas, gerir treinos, acompanhar progresso de membros | Desenhar programas de treino, liderar aulas, avaliar habilidades |
| **Personal Trainer** | Atribuir treinos, acompanhar progresso de membros | Sessões de treino pessoal, coaching individual |
| **Instrutor** | Liderar aulas atribuídas | Ensinar aulas de grupo, registrar presença |
| **Fisioterapeuta** | Ver dados de saúde, recuperação de membros | Avaliação de lesões, rastreamento de reabilitação |
| **Nutricionista** | Planos de dieta, consultas de membros | Orientação nutricional, planejamento de refeições |
| **Recepcionista** | Check-in/out, pagamentos, registro | Operações de balcão, assistência a membros |
| **Staff** | Acesso limitado, operações gerais | Funções de suporte, manutenção de instalações |

## Member Check-In | Check-In de Membros

### EN

#### Checking In a Member
1. Go to **Check-In** (or use Kiosk mode)
2. Search member by:
   - Scanning member ID card
   - Entering name
   - Entering phone number
3. Verify member details
4. Click **Check In**
5. System records:
   - Check-in time
   - Current location
   - Staff member who processed check-in

#### Checking Out a Member
1. Find checked-in member in **Check-In** list
2. Click **Check Out**
3. System records check-out time
4. Visit duration is calculated

#### Checking Membership Status
Before checking in, verify:
- **Active** status - Green indicator, allow check-in
- **Inactive** status - Red indicator, block check-in, inform member
- **Suspended** status - Orange indicator, check with manager
- **Pending** status - Yellow indicator, contact manager

### PT

#### Fazer Check-In de um Membro
1. Vá a **Check-In** (ou use modo Quiosque)
2. Pesquise membro por:
   - Escaneando cartão de ID de membro
   - Inserindo nome
   - Inserindo número de telefone
3. Verifique detalhes do membro
4. Clique em **Check-In**
5. Sistema registra:
   - Hora de check-in
   - Localização actual
   - Membro do staff que processou o check-in

#### Fazer Check-Out de um Membro
1. Encontre membro com check-in na lista **Check-In**
2. Clique em **Check-Out**
3. Sistema registra hora de check-out
4. Duração da visita é calculada

#### Verificar Estado de Associação
Antes de fazer check-in, verifique:
- Estado **Activo** - Indicador verde, permitir check-in
- Estado **Inactivo** - Indicador vermelho, bloquear check-in, informar membro
- Estado **Suspenso** - Indicador laranja, verificar com gerente
- Estado **Pendente** - Indicador amarelo, contactar gerente

## Class Management | Gestão de Aulas

### EN

#### For Coaches: Creating a Class
1. Go to **Classes** → **Add Class**
2. Fill in:
   - Title
   - Discipline (if applicable)
   - Workout template (if using a pre-defined workout)
   - Location
   - Date and time
   - Capacity
   - Is Mandatory? (for belt tests, assessments)
3. Click **Save**

#### Taking Attendance
Before or during class:
1. Open class details
2. Go to **Attendance** tab
3. For each member:
   - **Booked** - Confirmed attendance
   - **Checked-In** - Arrived and started class
   - **No-Show** - Did not attend
   - **Cancelled** - Cancelled before class
4. Click **Save Attendance**

#### Viewing Class Bookings
See who's booked for your class:
1. Open class details
2. Go to **Bookings** tab
3. View:
   - List of booked members
   - Waitlist members
   - Member rank information
   - Special notes

#### Managing Waitlist
If class is full:
1. View waitlist in class details
2. When someone cancels, system auto-promotes first waitlisted member
3. Member receives notification
4. Confirm manually if needed

#### For Instructors: Leading a Class
1. Open your assigned class
2. Review attached workout template (if any)
3. Take attendance as members arrive
4. During class:
   - Record any incidents
   - Note member performance
5. After class:
   - Save attendance
   - Add class notes if needed

### PT

#### Para Treinadores: Criar uma Aula
1. Vá a **Aulas** → **Adicionar Aula**
2. Preencha:
   - Título
   - Disciplina (se aplicável)
   - Modelo de treino (se usar um treino predefinido)
   - Localização
   - Data e hora
   - Capacidade
   - É Obrigatória? (para testes de faixa, avaliações)
3. Clique em **Salvar**

#### Registrar Presença
Antes ou durante a aula:
1. Abra os detalhes da aula
2. Vá à aba **Presença**
3. Para cada membro:
   - **Reservado** - Presença confirmada
   - **Com Check-In** - Chegou e começou a aula
   - **Não Compareceu** - Não participou
   - **Cancelado** - Cancelado antes da aula
4. Clique em **Salvar Presença**

#### Ver Reservas de Aulas
Veja quem reservou sua aula:
1. Abra os detalhes da aula
2. Vá à aba **Reservas**
3. Veja:
   - Lista de membros reservados
   - Membros na lista de espera
   - Informações de graduação do membro
   - Notas especiais

#### Gerir Lista de Espera
Se a aula estiver cheia:
1. Veja lista de espera nos detalhes da aula
2. Quando alguém cancelar, sistema promove automaticamente primeiro membro da lista
3. Membro recebe notificação
4. Confirme manualmente se necessário

#### Para Instrutores: Liderar uma Aula
1. Abra sua aula atribuída
2. Reveja modelo de treino anexado (se houver)
3. Registre presença enquanto membros chegam
4. Durante a aula:
   - Registre quaisquer incidentes
   - Note o desempenho dos membros
5. Após a aula:
   - Salve presença
   - Adicione notas de aula se necessário

## Workout Assignment | Atribuição de Treinos

### EN

#### Creating a Workout Template
1. Go to **Workouts** → **Templates**
2. Click **Create Template**
3. Fill in:
   - Template name
   - Description
   - Discipline (optional)
   - Difficulty level (beginner/intermediate/advanced/expert)
   - Min/Max rank level (for rank-based access)
   - Estimated duration
4. Add exercises:
   - Select from library or create new
   - Set reps/sets/duration
   - Add rest periods
5. Click **Save**

#### Assigning Workout to Member
1. Go to **Workouts** → **Assign**
2. Select member
3. Choose workout template
4. Set:
   - Assigned date
   - Due date (optional)
   - Notes for member
5. Click **Assign**

#### Viewing Assigned Workouts
1. Go to **My Assigned Workouts**
2. See list of members with assigned workouts
3. Track:
   - Pending workouts
   - Completed workouts
   - Member results and notes

#### For Trainers: Personal Training Sessions
When conducting personal training:
1. Assign specific workout to member
2. Meet for session
3. Guide member through workout
4. Record:
   - Weight used
   - Reps completed
   - Form corrections
   - Member feedback
5. Member submits completion with results

### PT

#### Criar um Modelo de Treino
1. Vá a **Treinos** → **Modelos**
2. Clique em **Criar Modelo**
3. Preencha:
   - Nome do modelo
   - Descrição
   - Disciplina (opcional)
   - Nível de dificuldade (iniciante/intermediário/avançado/experto)
   - Nível mínimo/máximo de graduação (para acesso baseado em graduação)
   - Duração estimada
4. Adicione exercícios:
   - Seleccione da biblioteca ou crie novo
   - Defina repetições/series/duração
   - Adicione períodos de descanso
5. Clique em **Salvar**

#### Atribuir Treino a Membro
1. Vá a **Treinos** → **Atribuir**
2. Seleccione membro
3. Escolha modelo de treino
4. Defina:
   - Data de atribuição
   - Data limite (opcional)
   - Notas para o membro
5. Clique em **Atribuir**

#### Ver Treinos Atribuídos
1. Vá a **Meus Treinos Atribuídos**
2. Veja lista de membros com treinos atribuídos
3. Acompanhe:
   - Treinos pendentes
   - Treinos concluídos
   - Resultados e notas dos membros

#### Para Personal Trainers: Sessões de Treino Pessoal
Ao conduzir treino pessoal:
1. Atribua treino específico ao membro
2. Encontre-se para sessão
3. Guie o membro pelo treino
4. Registre:
   - Peso usado
   - Repetições completadas
   - Correcções de forma
   - Feedback do membro
5. Membro envia conclusão com resultados

## Member Progress Tracking | Rastreamento de Progresso de Membros

### EN

#### For Coaches: Monitoring Skill Progress
1. Go to member profile
2. Go to **Progress** tab
3. View:
   - Current rank/belt in each discipline
   - Time in current grade
   - Attendance history
   - Workout completion rate
   - Personal records

#### For Coaches: Assessing Skills
When a member shows readiness for promotion:
1. Go to **Ranks** → **Schedule Assessment**
2. Create mandatory test class:
   - Set discipline
   - Date and time
   - Members being tested
   - Assessment criteria
3. Members book the mandatory class
4. Conduct assessment
5. Pass/Fail each member
6. **If passed:**
   - Promote to next rank
   - Certificate auto-generated
   - Member receives notification

#### For Physiotherapists: Tracking Recovery
1. Go to member profile
2. Go to **Health** tab
3. View health conditions and allergies
4. Add recovery notes:
   - Injury assessment
   - Treatment recommendations
   - Recovery exercises
   - Progress updates

#### For Nutritionists: Diet Planning
1. Go to member profile
2. Go to **Nutrition** tab (if available)
3. Create diet plan:
   - Meal recommendations
   - Calorie targets
   - Supplement suggestions
4. Track member adherence

### PT

#### Para Treinadores: Monitorar Progresso de Habilidades
1. Vá ao perfil do membro
2. Vá à aba **Progresso**
3. Veja:
   - Graduação/faixa actual em cada disciplina
   - Tempo na graduação actual
   - Histórico de frequência
   - Taxa de conclusão de treinos
   - Recordes pessoais

#### Para Treinadores: Avaliar Habilidades
Quando um membro demonstrar prontidão para promoção:
1. Vá a **Graduações** → **Agendar Avaliação**
2. Crie aula de teste obrigatória:
   - Defina disciplina
   - Data e hora
   - Membros sendo testados
   - Critérios de avaliação
3. Membros reservam a aula obrigatória
4. Conduza avaliação
5. Aprove/Reprove cada membro
6. **Se aprovado:**
   - Promova para próxima graduação
   - Certificado é gerado automaticamente
   - Membro recebe notificação

#### Para Fisioterapeutas: Acompanhar Recuperação
1. Vá ao perfil do membro
2. Vá à aba **Saúde**
3. Veja condições de saúde e alergias
4. Adicione notas de recuperação:
   - Avaliação de lesão
   - Recomendações de tratamento
   - Exercícios de recuperação
   - Actualizações de progresso

#### Para Nutricionistas: Planeamento Alimentar
1. Vá ao perfil do membro
2. Vá à aba **Nutrição** (se disponível)
3. Crie plano alimentar:
   - Recomendações de refeições
   - Metas calóricas
   - Sugestões de suplementos
4. Acompanhe adesão do membro

## Payment Processing | Processamento de Pagamentos

### EN - Receptionist Only

#### Processing a Payment
1. Go to **Payments** → **New Payment**
2. Select member
3. Enter:
   - Amount
   - Payment method:
     - **Multicaixa** - Ask for reference number, enter proof
     - **Cash** - Count cash, record
     - **Bank Transfer** - Upload receipt
     - **Other** - Enter details
4. Click **Process**

#### Verifying Multicaixa Payments
For Multicaixa:
1. Ask member to show proof (SMS, app screenshot)
2. Verify reference number matches
3. Enter reference number in payment
4. Upload proof image
5. Mark as verified

#### Handling Refunds
1. Go to **Payments**
2. Find payment to refund
3. Click **Refund**
4. Enter:
   - Refund amount
   - Reason
5. Confirm refund

#### Invoicing
When member needs invoice:
1. Go to **Payments** → **Invoices**
2. Create invoice:
   - Select member
   - Add line items
   - Set due date
3. Click **Generate**
4. Member can view in their portal

### PT - Apenas Recepcionista

#### Processar um Pagamento
1. Vá a **Pagamentos** → **Novo Pagamento**
2. Seleccione membro
3. Insira:
   - Valor
   - Método de pagamento:
     - **Multicaixa** - Peça número de referência, insira comprovativo
     - **Dinheiro** - Conte dinheiro, registre
     - **Transferência Bancária** - Faça upload do comprovativo
     - **Outros** - Insira detalhes
4. Clique em **Processar**

#### Verificar Pagamentos Multicaixa
Para Multicaixa:
1. Peça ao membro para mostrar comprovativo (SMS, screenshot do app)
2. Verifique se o número de referência corresponde
3. Insira número de referência no pagamento
4. Faça upload da imagem do comprovativo
5. Marque como verificado

#### Lidar com Reembolsos
1. Vá a **Pagamentos**
2. Encontre pagamento para reembolsar
3. Clique em **Reembolsar**
4. Insira:
   - Valor do reembolso
   - Motivo
5. Confirme reembolso

#### Facturação
Quando membro precisar de factura:
1. Vá a **Pagamentos** → **Facturas**
2. Crie factura:
   - Seleccione membro
   - Adicione itens de linha
   - Defina data de vencimento
3. Clique em **Gerar**
4. Membro pode ver no seu portal

## Activity Reporting | Relatórios de Atividade

### EN

#### View Your Activity
1. Go to **My Activity**
2. See:
   - Today's check-ins processed
   - Classes you taught
   - Workouts you assigned
   - Payments you processed (if receptionist)

#### Submitting Reports
For incidents, issues, or concerns:
1. Go to **Reports** → **New Report**
2. Fill in:
   - Type (incident, facility issue, member concern)
   - Description
   - Date and time
   - Affected member(s)
3. Click **Submit**
4. Manager receives notification

### PT

#### Ver Sua Atividade
1. Vá a **Minha Atividade**
2. Veja:
   - Check-ins processados hoje
   - Aulas que você ensinou
   - Treinos que você atribuiu
   - Pagamentos que você processou (se recepcionista)

#### Submeter Relatórios
Para incidentes, problemas ou preocupações:
1. Vá a **Relatórios** → **Novo Relatório**
2. Preencha:
   - Tipo (incidente, problema de instalações, preocupação de membro)
   - Descrição
   - Data e hora
   - Membro(s) afectado(s)
3. Clique em **Submeter**
4. Gerente recebe notificação

## Tips for Success | Dicas para Sucesso

### EN

#### For Coaches/Trainers
- Always review member health notes before workouts
- Update member progress regularly
- Communicate clearly about rank requirements
- Use workout templates for consistency
- Document member achievements

#### For Receptionists
- Be patient and helpful with check-ins
- Verify payment references carefully
- Keep reception area clean and organized
- Know member names for personal service
- Escalate issues to manager promptly

#### For Instructors
- Arrive early to set up equipment
- Introduce yourself to new members
- Keep classes engaging and safe
- Record attendance accurately
- Give feedback to members

#### For Physiotherapists
- Document assessments thoroughly
- Communicate with coaches about limitations
- Follow up on recovery progress
- Keep health data confidential
- Coordinate return-to-activity plans

### PT

#### Para Treinadores/Personal Trainers
- Sempre reveja as notas de saúde dos membros antes dos treinos
- Actualize o progresso dos membros regularmente
- Comunique claramente sobre requisitos de graduação
- Use modelos de treino para consistência
- Documente conquistas dos membros

#### Para Recepcionistas
- Seja paciente e útil com check-ins
- Verifique referências de pagamento com cuidado
- Mantenha a área de recepção limpa e organizada
- Conheça os nomes dos membros para serviço pessoal
- Escale problemas para gerente prontamente

#### Para Instrutores
- Chegue cedo para configurar equipamento
- Apresente-se aos novos membros
- Mantenha aulas envolventes e seguras
- Registre presença com precisão
- Dê feedback aos membros

#### Para Fisioterapeutas
- Documente avaliações minuciosamente
- Comunique com treinadores sobre limitações
- Faça acompanhamento do progresso de recuperação
- Mantenha dados de saúde confidenciais
- Coordene planos de retorno à actividade
